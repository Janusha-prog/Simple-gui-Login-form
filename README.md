# Simple-gui-Login-form
A beginner-friendly GUI Login Form built using Python and Tkinter that allows users to log in with multiple username and password combinations through a simple graphical interface.

# Features
- User login interface
- Password hidden using '*'
- Multiple user login support
- Error handling for invalid login
- Simple and beginner-friendly GUI

# Technologies Used
- Python
- Tkinter
